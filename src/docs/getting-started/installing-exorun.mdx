# Installing Exorun-cli
Exorun-cli is a command-line utility that lets you work with Exorun, from creating your account to deploying your applications. It runs on your local device so you'll want to install the version that's appropriate for your operating system:

[macOS](#macos)

[Linux](#linux)

[Windows](#windows)

## macOS

### With a Script
```sh
$ curl -L https://getexorun.exorun.dev/install.sh | sh
```


## Linux

### With a Script
```sh
$ curl -L https://getexorun.exorun.dev/install.sh | sh
```

## Windows

Download the executable binary exorun-win.exe using the curl command:
```sh
$ curl -LO https://getexorun.exorun.dev/windows/exorun-win.exe
```
